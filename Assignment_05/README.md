# Assignment 05 - Spark Fundamentals and Data Processing using PySpark

## Objective

The objective of this assignment is to understand Apache Spark fundamentals and perform data cleaning, transformation, aggregation, and schema modification using PySpark DataFrames. The assignment also demonstrates Spark's in-memory processing, DataFrame immutability, filtering, grouping, and building a complete data processing pipeline.

## Technologies Used

- Python 3.10
- Apache Spark (PySpark)
- Jupyter Notebook

## Files Included

- `spark_assignment.ipynb` – PySpark notebook containing all solutions, code, and outputs.
- `brief_insights.txt` – Key observations and insights from the assignment.
- `Screenshots/` – Screenshots of code execution and query outputs.
- `README.md` – Project documentation.

## Tasks Performed

- Studied the limitations of MapReduce and the advantages of Apache Spark.
- Learned Spark DataFrame concepts and immutability.
- Removed duplicate records from a DataFrame.
- Handled missing and null values using PySpark.
- Applied filtering conditions on datasets.
- Performed aggregation using `count()`, `sum()`, `avg()`, `min()`, and `max()`.
- Grouped data using `groupBy()`.
- Understood shuffle operations and wide transformations.
- Modified DataFrame schema by casting and renaming columns.
- Handled inconsistent data such as null and empty values.
- Built a complete data processing pipeline combining data cleaning and aggregation.

## Learning Outcomes

After completing this assignment, I gained practical experience in:

- Working with PySpark DataFrames.
- Performing data cleaning and preprocessing.
- Applying DataFrame transformations and filtering.
- Using aggregation functions for data analysis.
- Understanding Spark execution concepts such as immutability and shuffle.
- Building an end-to-end data processing pipeline using Apache Spark.
