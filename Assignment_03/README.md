# Advanced SQL Analysis using Superstore Dataset

## Overview

This project was completed as part of the **Celebal Technologies Data Engineering Internship (Assignment 03)**.

The objective of this assignment is to analyze the Superstore dataset using advanced SQL concepts such as Subqueries, Common Table Expressions (CTEs), Window Functions, and Joins to solve business-oriented analytical problems.

---

## Dataset

**Dataset Used:** Superstore Dataset

The dataset contains information related to:

- Customers
- Orders
- Products
- Sales
- Profit
- Discounts
- Shipping Details

## Business Questions Solved

- Identify the Top 5 Customers based on Total Sales.
- Identify the Bottom 5 Customers based on Total Sales.
- Find Customers Who Placed Only One Order.
- Identify Customers with Above Average Total Sales.
- Find the Highest Order Value for Each Customer.

## Key Outcomes

- Created normalized tables for Customers, Orders, and Products.
- Applied advanced SQL techniques for analytical reporting.
- Generated customer rankings using Window Functions.
- Performed business-oriented sales analysis.
- Produced meaningful business insights from the Superstore dataset.
