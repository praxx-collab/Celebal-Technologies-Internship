# E-Commerce Sales Database Analysis using MySQL

## Objective

The objective of this project is to design and analyze an E-Commerce Sales Database using MySQL. The project demonstrates SQL fundamentals, data retrieval, aggregation, joins, filtering, CASE statements, and transaction management.

---

## Database

**Database Name:** `ecommerce_db`

### Tables Created

* Customers
* Products
* Orders
* Order_Items

---

## Topics Covered

* Database and Table Creation
* Primary Key & Foreign Key Constraints
* Data Insertion
* SELECT Queries
* Filtering using WHERE
* Aggregation (COUNT, SUM, AVG, MIN, MAX)
* GROUP BY & HAVING
* INNER JOIN & LEFT JOIN
* CASE Statements
* ACID Properties
* SQL Transactions (COMMIT & ROLLBACK)

---

## Files Included

* `ecommerce_database.sql` – Complete SQL script with all queries.
* `Ecommerce_Assignment.ipynb` – Jupyter Notebook containing SQL queries, outputs, and explanations.

* Designed a relational database.
* Practiced writing SQL queries for business scenarios.
* Applied joins, aggregation, filtering, and transactions.
