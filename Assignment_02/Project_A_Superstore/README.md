## Project Overview

This project demonstrates SQL-based data analysis on the Sample Superstore dataset using MySQL. The analysis includes data exploration, filtering, aggregation, sorting, business queries, and data validation.

## Dataset

- Sample Superstore Dataset
- Total Records: 9694

## Tools Used

- MySQL Workbench
- Visual Studio Code
- Jupyter Notebook
- Python (Pandas, MySQL Connector)

## Project Files

- `ecommerce_sales.sql` - SQL queries used for analysis
- `SQL_Analysis.ipynb` - Jupyter Notebook containing SQL execution and results
- `Business Insights.txt` - Summary of key business insights


## SQL Concepts Covered

- SELECT
- WHERE
- GROUP BY
- ORDER BY
- LIMIT
- Aggregate Functions
- Data Validation
- Business Analysis

## Key Insights

- West region generated the highest total sales.
- Technology was the highest revenue-generating category.
- Office Supplies recorded the highest quantity sold.
- Home Office customers had the highest average sales.
- Sean Miller was the highest-value customer.
- Canon imageCLASS 2200 Advanced Copier was the top-selling product.
- No NULL values were found in the Sales and Profit columns.

## Outcome

The project successfully demonstrates SQL data analysis techniques for extracting meaningful business insights from an e-commerce sales dataset.
