**Azure Cloud Fundamentals and Data Pipeline Implementation using Azure Data Factory**

*Objective:*



To understand the basic concepts of Microsoft Azure Cloud and implement a complete data pipeline using Azure Storage Account and Azure Data Factory (ADF).



*Work Performed:*

1\. Explored the Azure Portal and created a Resource Group.

2\. Created an Azure Storage Account and a Blob Container.

3\. Uploaded the Sample - Superstore.csv file to Azure Blob Storage.

4\. Created an Azure Data Factory instance.

5\. Configured a Linked Service to connect Azure Data Factory with Azure Blob Storage using an Account Key.

6\. Created Source and Destination datasets.

7\. Developed a pipeline using Get Metadata and Copy Data activities.

8\. Validated and published the pipeline.

9\. Executed the pipeline successfully using Debug.

10\. Verified that the file was copied successfully to the destination as Output\_Superstore.csv.



*Azure Services Used:*

1\. Microsoft Azure Portal

2\. Resource Group

3\. Azure Storage Account

4\. Azure Blob Storage

5\. Azure Data Factory

6\. Linked Service

7\. Dataset

8\. Get Metadata Activity

9\. Copy Data Activity



*Outcome:*



The Azure Data Factory pipeline was successfully executed. The source CSV file stored in Azure Blob Storage was processed, metadata was validated using the Get Metadata activity, and the file was copied successfully to the destination location. The output file (Output\_Superstore.csv) was generated successfully, demonstrating a complete end-to-end Azure data pipeline.
